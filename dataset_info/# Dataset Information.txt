# Dataset Information

## Dataset Name

Brazilian E-Commerce Public Dataset by Olist

## Source

Kaggle

## Description

The dataset contains information about orders placed on the Olist Store platform in Brazil.

## Tables Used

1. Customers
2. Orders
3. Order Items
4. Payments
5. Reviews
6. Products
7. Sellers
8. Geolocation
9. Product Category Translation

## Key Fields

### Customers

* customer_id
* customer_city
* customer_state

### Orders

* order_id
* customer_id
* order_status
* order_purchase_timestamp

### Payments

* payment_type
* payment_value

### Reviews

* review_score

## Use Case

The dataset was used to build an end-to-end Data Engineering solution using Microsoft Fabric, PySpark, Lakehouse Architecture, Power BI, and Data Pipelines.
